Admin Role

Administrators have complete access to all features in the system, including user management, system configuration, and all trading capabilities.

Use this role for: System administrators and supervisors who need to manage users and have complete control over the system.

User Role

Standard users have access to trading features and can manage their own account settings, but cannot access administrative functions.

Use this role for: Traders who need to execute trades and manage their trading settings.

Viewer Role

Viewers have read-only access to the system. They can view dashboard statistics, trades, and signals, but cannot make any changes.

Use this role for: Observers who need to monitor system activity without making changes.

Role Assignment Guidelines
Best Practices
  • Assign the Admin role sparingly and only to trusted personnel
  • Use the User role for most active traders
  • Use the Viewer role for observers and analytics-only users
  • Periodically review user roles to ensure appropriate access
Security Considerations
  • Admin users can create other admin users
  • Admin users have access to sensitive system settings
  • Users can execute trades that involve financial risk
  • Viewers cannot make changes but can see sensitive data
